# Shorts WhatsApp SaaS

MVP para generar videos verticales cortos orientados a negocios que venden por WhatsApp.

## Estructura

- `apps/web`: landing de alta conversion + flujo de generacion.
- `apps/api`: API base para generar previews, regenerar y listar planes.
- `docs`: contrato y contexto del producto.

## Inicio rapido

1. `npm install`
2. `npm run dev:api`
3. `npm run dev:web`
